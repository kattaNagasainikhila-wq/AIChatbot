# 🤖 Rule-Based AI Chatbot

## 📌 Project Overview

This project is a Rule-Based AI Chatbot developed using Python as part of my AI Internship at DecodeLabs.

The chatbot interacts with users using predefined rules and responses. It processes user input, performs basic decision-making, and responds accordingly.

---

## 🚀 Features

✅ Greeting and conversation handling

✅ Input sanitization using lower() and strip()

✅ Continuous interaction loop

✅ Date and time functionality

✅ Joke generation

✅ Help command

✅ Exit command

✅ Fallback response for unknown inputs

## 🛠️ Technologies Used

- Python
- Dictionaries (Hash Maps)
- Control Flow
- Loops
- Conditional Statements
